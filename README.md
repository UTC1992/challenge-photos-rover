## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000/gallery/rover](http://localhost:3000/gallery/rover) with your browser to see the result.

## Run test

```bash
yarn test
# or
yarn test:watch
# or
npm run test
# or
npm run test:watch
```
